# Todo - App
A fancy todo app which allows you to manage your task efficiently

---
## Getting Started
copy this project into your local machine
```
npm install
```

## Tech Stack
- React
- Redux
- React-hook-form
- Material-UI
- Node.js
- Express
- MongoDB
- Mongoose
- JsonWebToken
